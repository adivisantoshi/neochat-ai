# NeoChat AI 🌌

A futuristic cyberpunk AI chat application with streaming responses, neon UI, microphone input, and a fully custom frontend built using Next.js, FastAPI, and Ollama.

---

## ✨ Features

- ⚡ Real-time streaming AI responses
- 🌌 Cyberpunk neon UI
- 🎤 Voice input support
- 💖 Electric pink animations
- 🧠 Local LLM integration using Ollama
- 🖥️ FastAPI backend
- ⚛️ Next.js frontend
- 🌊 Smooth transitions and animations
- ✨ Custom cursor trail effects
- 📱 Responsive design

---

## 🛠️ Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python
- Ollama
